# Security Policy

This is a small, single-purpose, security-relevant library. If you find a vulnerability, please report it
**privately** — do not open a public issue.

- Preferred: this repo's **Security** tab → **Report a vulnerability** (a private GitHub advisory).
- Or email **isaac@pantheonlabs.co.uk**.

Please include a description, reproduction steps, and the impact. I'll acknowledge as soon as I can and work with
you on a fix and coordinated disclosure. Maintained on a best-effort basis (no formal SLA), but security reports
are taken seriously.
